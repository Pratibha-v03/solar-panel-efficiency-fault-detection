## Project Overview

This project focuses on **monitoring solar panel efficiency** and **detecting faults** in solar energy systems using data analytics, machine learning, and visualization techniques. Completed during a Data Analytics Internship at **Solar Secure Solutions**, where real-world solar sensor and API data was leveraged to build end-to-end analytical pipelines.

---

## Key Contributions

### 1. Data Collection & Cleaning

- Collected and cleaned solar datasets from **IoT sensors** and **external APIs** using **Python**, **NumPy**, and **Pandas**
- Applied rigorous **data quality checks** including null handling, outlier removal, and schema validation
- Ensured data integrity across multi-source pipelines for consistent downstream analysis

### 2. Exploratory Data Analysis (EDA)

- Performed comprehensive **EDA in Jupyter Notebook**, uncovering patterns and anomalies in solar energy data
- Achieved **90% accuracy** in solar energy generation insights through statistical analysis
- Identified correlations between irradiance, temperature, and panel output degradation

### 3. Dashboards & Visualizations

- Created interactive **dashboards with Power BI** for real-time energy output monitoring
- Built analytical charts using **Matplotlib** and **Seaborn** for maintenance diagnostics
- Delivered visual reports on energy performance trends and fault indicators to stakeholders

### 4. Machine Learning Pipelines

- Built **ML-based monitoring pipelines** for solar system performance tracking using trend analysis and fault detection
- Applied **fault detection algorithms** to flag underperforming panels and track efficiency metrics
- Enabled **predictive maintenance** by identifying efficiency drops and anomalous behavior patterns
